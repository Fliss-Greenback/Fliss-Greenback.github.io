ALWAYS CHECK THE FILEPATH IS CORRECT!

Learning and Growing, Jan 12, 2021

If something is not working as expected

Always check the filepath is correct - start with syntax ../ to check which directory you are in.

This will save you many hours of headscratching to figure out the solution to a non-problem.
